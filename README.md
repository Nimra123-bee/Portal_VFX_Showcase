# Portal VFX Showcase

This repository showcases stylized portal effects developed during my internship at Codematics, designed to function as living environmental elements rather than static visuals.

The goal was to create portals that feel active, dynamic, and integrated into the game world.

## 🎯 Design Focus

The primary objective was to make portals feel alive.

This was achieved by:
- creating animated inner cores using distortion  
- adding continuous particle motion flowing in and out of the portal  
- designing outer rings with rotational energy patterns  

Both the center and outer structure were treated as active components rather than decorative elements.

---

## 🧠 My Contribution

- Designed portal effects with dynamic inner and outer behavior  
- Created animated core visuals using Shader Graph distortion  
- Built particle systems to simulate energy flow through portals  
- Applied rotation over lifetime to create continuous motion in outer rings  
- Balanced visual density and motion for stable looping effects  

---

## ⚙️ Tools & Techniques

- Unity Particle System (motion, emission, looping systems, noise etc)  
- Shader Graph (distortion-based inner visuals)  
- Post Processing (visual enhancement)  

---

## 🧩 Key Challenge

One of the main challenges was working with Shader Graph without prior experience.

This required:
- learning shader basics during development  
- experimenting with distortion techniques  
- iterating on visual feedback to achieve a stable and believable result
